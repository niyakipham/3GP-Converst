3GP CONVERST - VIDEO TO 3GP CONVERTER

3GP CONVERST is a bash script that converts MP4 videos to 3GP format (the classic mobile phone video format).

Key Features:

  • Converts all MP4 videos to 3GP format
  • Automatically names output files sequentially (e.g., 1.3gp, 2.3gp, etc.)

Relive the nostalgia of early mobile video formats with this efficient conversion tool.

INSTALL

For Ubuntu:

sudo apt install ffmpeg

For Arch Linux:

sudo pacman -S ffmpeg

UPDATE

  • Reduces frame rate from 20 fps to 15 fps
  • Reduces resolution
  • Reduces video bitrate
  • Reduces video quality

Technical Details:

# 1. Loop through all .mp4 files in current directory
for file in *.mp4; do
    
    # 2. Check if file is .mp4
    if [[ -f "$file" && "$file" == *.mp4 ]]; then
        
        # 3. Create output filename
        output_file="$output_dir/$counter.3gp"
        
        # 4. Convert .mp4 to .3gp using ffmpeg
        ffmpeg -y -i "$file" \
            -r 20 -s 352x288 -vb 400k \
            -acodec aac -strict experimental -ac 1 -ar 8000 -ab 24k \
            "$output_file"
            
        # 5. Increment counter for sequential naming
        ((counter++))
    fi
done

RUN

  1. Move all videos you want to convert into a single directory
  2. Copy the 3gp.sh executable file into that directory
  3. Run the file
  4. All MP4 videos will be converted to 3GP format in seconds

The conversion process is optimized for speed while maintaining compatibility with older devices.

NOTE

IMPORTANT:

The output files may be LARGER in size than the input files. Before conversion, it's recommended to download videos at 144p quality to avoid excessive file sizes.
